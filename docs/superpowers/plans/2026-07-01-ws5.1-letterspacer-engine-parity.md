## Reference algorithm (what the original does, condensed)

`HTLSEngine.setSpace` (engine.py:413), per side:

1. `totalMarginList` samples the **full glyph height** `[layer.minY, layer.maxY]` in steps of `freq`. Every scanline yields a point: the real outline margin, or — when a scanline has no outline — a **slant-default** point (`origin + triangle(angle, y) + dfltDepth` on the left, `origin + triangle(angle, y)` on the right). This makes the margin list a dense, contiguous function of `y`.
2. `zoneMargins` filters that list to the reference zone `[minYref, maxYref]`.
3. If the master is italic, `deslant` both the zone and full margins.
4. `maxPoints` gives the **full** extremes (`lFullExtreme`, `rFullExtreme`) and the **zone** extremes (`lExtreme`, `rExtreme`).
5. `processMargins` on the zone margins: `setDepth` (clip to `extreme ± depth`, then **pad** the list up/down to fill the whole zone at max depth) → `diagonize` (clamp x-steps to ±`ystep` so counters close at 45°) → `closeOpenCounters` (prepend/append the extreme corners → closed polygon).
6. `distanceL = ceil(lExtreme - lFullExtreme)`, `distanceR = ceil(rFullExtreme - rExtreme)` — how far the true horizontal extreme sits outside the x-height zone.
7. `calculateSBValue(polygon) = (propArea - area(polygon)) / amplitudeY`, where `propArea = amplitudeY * (effectiveArea * (upm/1000)^2 * 100) / xHeight`.
8. `newL = ceil(-distanceL + SBValue(lPolygon))`, `newR = ceil(-distanceR + SBValue(rPolygon))`.

Our current `computeSpacing` implements 7 correctly but skips 1's fill, 5's padding + diagonize, 3's deslant, 6's correction, and 8's ceil. This plan adds them.

```js
// Close open counterforms at 45°: clamp each consecutive x-step to ±ystep,
// forward then reverse. Verbatim port of engine.py diagonize (358-373).
export function diagonize(marginsL, marginsR, ystep) {
  const L = marginsL.map((p) => ({ ...p }));
  const R = marginsR.map((p) => ({ ...p }));
  for (let i = 0; i < L.length - 1; i++) {
    if (L[i + 1].x - L[i].x > ystep) {
      L[i + 1].x = L[i].x + ystep;
    }
    if (R[i + 1].x - R[i].x < -ystep) {
      R[i + 1].x = R[i].x - ystep;
    }
  }
  for (let i = L.length - 2; i >= 0; i--) {
    if (L[i].x - L[i + 1].x > ystep) {
      L[i].x = L[i + 1].x + ystep;
    }
    if (R[i].x - R[i + 1].x < -ystep) {
      R[i].x = R[i + 1].x - ystep;
    }
  }
  return [L, R];
}
```

```js
// Pad the margin list up and down so it spans the whole reference zone at the
// depth-limit x. Verbatim port of the loops in engine.py setDepth (344-354).
export function padMarginsToZone(
  marginsL,
  marginsR,
  leftDepthLimit,
  rightDepthLimit,
  minYref,
  maxYref,
  freq
) {
  const L = marginsL.map((p) => ({ ...p }));
  const R = marginsR.map((p) => ({ ...p }));
  if (L.length === 0) {
    return [L, R];
  }
  let y = L[0].y - freq;
  while (y > minYref) {
    L.unshift({ x: leftDepthLimit, y });
    R.unshift({ x: rightDepthLimit, y });
    y -= freq;
  }
  y = L[L.length - 1].y + freq;
  while (y < maxYref) {
    L.push({ x: leftDepthLimit, y });
    R.push({ x: rightDepthLimit, y });
    y += freq;
  }
  return [L, R];
}
```

```js
// ceil'd units (engine.py setSpace 454-455). Subtracted from the SB value so
// protrusions outside the zone do not collide.
export function sidebearingCorrection(zoneExtreme, fullExtreme, side) {
  if (side === "left") {
    return Math.ceil(zoneExtreme - fullExtreme);
  }
  return Math.ceil(fullExtreme - zoneExtreme);
```

```js
    this.leftMargins = margins.leftMargins.filter(

    this.rightMargins = margins.rightMargins.filter(
      (p) => p.y >= refMinY && p.y <= refMaxY
    );

    if (this.leftMargins.length < 2 || this.rightMargins.length < 2) {
      return { lsb: null, rsb: null };
    }

    // Zone extremes drive the depth limits (engine.py maxPoints over the zone).
    this.leftExtreme = Math.min(...this.leftMargins.map((p) => p.x));
    this.rightExtreme = Math.max(...this.rightMargins.map((p) => p.x));
```

```js
// De-slant a point list to upright, origin at half the x-height
// (engine.py deslant 383-389). Identity when angle is 0.
export function deslant(margins, angle, xHeight) {
  if (!angle) {
    return margins.map((p) => ({ ...p }));
  }
  const mline = xHeight / 2;
  const t = Math.tan((angle * Math.PI) / 180);
  return margins.map((p) => ({ x: p.x - (p.y - mline) * t, y: p.y }));
}

// Re-slant a point list back to the italic angle (engine.py slant 391-397).
export function slant(margins, angle, xHeight) {
  return deslant(margins, -angle, xHeight);
}
```
